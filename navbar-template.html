Page Template

This shows how to use the reusable navbar component

How to use:

  1. Add <div id="navbar-container"></div> where you want the navbar
  2. Include the navbar script: <script src="assets/js/navbar.js"></script>
  3. That's it! The navbar will automatically load and highlight the current page

✨ Benefits:

  • Change navigation links in one place (assets/js/navbar.js)
  • All pages automatically update
  • Active page highlighting works automatically
  • Consistent styling across all pages
  • Easy to maintain and update
